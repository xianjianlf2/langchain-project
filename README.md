### start up

```bash
# server
cd ./server
pnpm dev

# frontend
cd ..
pnpm dev
```

### config

```bash
// server/.env
PINECONE_API_KEY="YOUR-KEY"
PINECONE_ENVIRONMENT="YOUR-KEY"
PINECONE_INDEX="YOUR-KEY"
OPENAI_API_KEY="YOUR-KEY"
OPENAI_PROXY_URL="YOUR-KEY"
```
